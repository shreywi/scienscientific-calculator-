# Scientific Calculator

A modern, responsive scientific calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: `+`, `-`, `*`, `/`
- Scientific functions: `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `log`, `ln`, `sqrt`, `abs`, `exp`
- Power and factorial: `^`, `!`
- Constants and extras: `pi`, `%`, parentheses
- Angle mode toggle: `DEG` and `RAD`
- Memory controls: `MC`, `MR`, `M+`, `M-`
- Reusable history panel
- Keyboard shortcuts:
  - `Enter` to evaluate
  - `Backspace` to delete last character
  - `Esc` to clear all

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/scientific-calculator.git
   ```
2. Open the project folder.
3. Open `index.html` in your browser.

## Publish to GitHub

1. Initialize Git (if needed):
   ```bash
   git init
   git add .
   git commit -m "Initial scientific calculator project"
   ```
2. Create a new repository on GitHub named `scientific-calculator`.
3. Link and push:
   ```bash
   git remote add origin https://github.com/<your-username>/scientific-calculator.git
   git branch -M main
   git push -u origin main
   ```

## Optional: Enable GitHub Pages

1. Go to **Settings > Pages** in your repository.
2. Set source to `Deploy from a branch`.
3. Select `main` branch and `/ (root)` folder.
4. Save, then open the generated GitHub Pages URL.
